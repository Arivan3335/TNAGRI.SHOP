Tailwind-Framework for Wordpress (Fast & Optimized),Unparalleled Versatility, Advanced Elementor & Gutenberg integration and etc...


== Description ==
By adding the Tailwind framework to your WordPress in a completely dynamic way, you can design and style all parts of your site using this powerful framework, this plugin is also a new and suitable platform for developing WordPress Themes in a more optimal & faster way. This plugin also is compatible with the default WordPress page builder, Gutenberg, and the very popular Elementor page builder.


== Installation ==
To install the plugin, just search the term `Taily` or `Tailwind` in the wordpress repository!

== Uninstalling ==
Before deleting the plugin, you can delete all the information of this plugin in your site's database forever by activating the `Clearing while Uninstall` option when deleting.
